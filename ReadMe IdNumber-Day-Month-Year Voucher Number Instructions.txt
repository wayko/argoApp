File naming convention instructions

Example: 823383-3-17-2017 609_10110232-5BDE38229C.pdf

Student ID Number + dd-mm-yyyy Original Voucher Number

Student ID: type the student ID 
dd-mm-yyyy date: in US format dd = day of the month, mm = Month, and YYYY = full year number
Space: leave a blank space
Original Voucher Number: do not delete the Original file name.

This procedure will help to automate the vouchers in the future

Lead Amil know if you prefer other formats.  We would like to have a single standard. the Voucher number should appear in the filename.
